
Installation:

Copy the file "pt_EffectSearch.jsx" into the following folder:

Windows: C:\Program Files\Adobe\Adobe After Effects C[xx]\Support Files\Scripts\ScriptUI Panels

Mac: /Applications/After Effects C[xx]/Scripts/ScriptUI Panels

You should allow scripts to write files in After Effects:
   Edit->Preferences->General->Allow Scripts to Write Files and Access Network
   


Установка:

Скопируйте "pt_EffectSearch.jsx" в следующее место в каталоге установки AE:

Windows: C:\Program Files\Adobe\Adobe After Effects C[xx]\Support Files\Scripts\ScriptUI Panels

Mac: /Applications/After Effects C[xx]/Scripts/ScriptUI Panels

Поставьте галочку в Edit->Preferences->General->Allow Scripts to Write Files and Access Network
