pt_TextEdit v2 Installation Notes.

This script requires After Effects CS4 or later.

NOTE: The script can be installed as either a floating window or dockable panel.

INSTALL AS DOCKABLE PANEL:
- Install the script in After Effects/Scripts/ScriptUI Panels (MAC) or After Effects/Support Files/Scripts/ScriptUI Panels (WIN).
- Restart AE and select the script from the bottom of the Window menu.

INSTALL AS FLOATING WINDOW:
- Install the script in After Effects/Scripts/ (MAC) or After Effects/Support Files/Scripts/ (WIN).
- Restart AE and select the script from the File -> Scripts menu.
