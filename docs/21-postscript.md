# 21 后记

> 不忧愁的脸，是我的少年。不仓惶的眼，等岁月改变。    ——【犬夜叉二十周年·原创】穿越时空的思念（BV1Ls411x7BX）

天啊，终于仿完了。这可能是读者你此时的感觉。而对我而言是，天啊，终于写完这个教程了。

实际上，这个教程是我一怒之下的结果。因为我不理解为何这样一个教科书级别的MAD时隔12年依旧没有一个完善的教程，于是我动手了。在写完教程的此刻，我突然释然了，毕竟MAD文化太小众，但这不影响我对技术的追求以及分享。事实上，我希望也有人能认真写下类似《灭星蚀》这类MAD作品的完整教程。

至此，2010秋季OP这个作品已经基本模仿完成。动画基地这个刊物，虽然已经成为时代的眼泪，但是它没有逝去，因为有不少人依旧记得它，依旧记得那个很好的年代，以及我们只能存留在回忆中的童年。时光飞逝啊，恍然之间，就这样长大了。

动漫刊物纸媒体的衰落是时代发展的结果，谁也无法阻止，就正如谁也无法阻止当今视频网站文化的兴起和繁荣。

本教程到此结束。感谢你能阅读这个教程。
