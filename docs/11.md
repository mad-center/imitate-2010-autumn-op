## 人物 actors

新建一个合成actors，在合成actors里面导入人物素材。在外层合成中，对该合成制作一个缩小的动画。

![image-20220126163700027](assets/image-20220126163700027.png)

---

因为在时间线末尾需要将人物转化为填充模式，因此可以复制一层actors放于上方，添加填充。然后在合适时间点左裁剪。

![image-20220126164025494](assets/image-20220126164025494.png)

---

接着，为了实现人物右偏移的效果。复制一层actors放于上方。添加transform变换。

- 将 transform 效果中position的X设置为660。也就是相对中心点在X方向右偏移20 pixel。

- 然后对opacity进行闪动k帧。参考值：

  - 0 50 0
  - 0 35 35 0
  - 0 35 35 0
  - 0 70 0

  为了使得opacity的变动更加真实随机，对opacity进行轻度wiggle。参考：`wiggle(3,10)`。

![image-20220126164149028](assets/image-20220126164149028.png)

---

上面已经完成了偏移效果，最后还需要一个横线的故障效果叠加。这个效果通过brush strokes来实现。

![image-20220126164904162](assets/image-20220126164904162.png)

- 复制opacity wiggle这个图层，去掉之前opacity的关键帧，并使用表达式将opacity变化绑定到opacity wiggle图层。
- 对该图层 actors linearize添加brush strokes效果，将角度设置为水平方向，并使用表达式time对笔刷“描边随机”属性进行演化。

## 过渡矩形

### bars-transition

![image-20220126170458809](assets/image-20220126170458809.png)

- 首先确定过渡开始的时间点，图中【2】标记处。
- 使用形状层，绘制一个矩形。
  - 将初始位置关键帧定于画布左侧不可见区域，
  - 将中间位置关键帧定于画布可见区域，
  - 将结束位置关键帧定于画布右侧不可见区域。

- 提升中间关键帧的速度值点，必须保证>0。

  ![image-20220126170750691](assets/image-20220126170750691.png)

根据这个方法，创建多个矩形形状图层。尽可能布满画布空间。

由于图层数量很多，在创建副本之前，务必思考矩形阵列的分布。对于初始化参数的设计，这里给出一个参考。

- 大小初始化：1280x100

- 位置初始化：X位于[-900,-1400] Y位于[100,800]之间

- scale的X为一个固定值，例如80%，Y位于50%-100%之间波动。

另外，注意稍微错开矩形形状层的动画起始时间。

![image-20220126171121449](assets/image-20220126171121449.png)

---

最后，创建2个文本。并挑选一个合适的时间点，将文本图层的位置绑定到其中2个矩形图层。

> 可以通过调整文本图层的锚点位置来控制文本图层在矩形图层中的位置偏移。这步属于微调文本位置。
