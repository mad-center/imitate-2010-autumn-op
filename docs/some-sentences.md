# 一些句子

- 世外悠悠隔人间，不忍凄凄乱世烟。
- 寄蜉蝣于天地，渺沧海之一粟。
- 岁月流年花不复，若水韶光樱飞时。
- 隙间白玉，花开满庭，墨染樱轮，西行幽幽
- 音无结弦之时，跃动天使之心。立于浮华之世，奏响天籁之音。——《Angel beats》

- 樱花满地集于我心，蝶舞纷飞祈愿相随。——《罪恶王冠》

- 雪中智代雨中杏，树下古河花田汐。——《Clannad》

- 如梦落樱满熏香，梦醒犹记四月谎。——《四月是你的谎言》

- 梦断青山，心坠七海。——《樱花庄的宠物女孩》

- 木叶飞舞之处，火亦生生不息。——《火影忍者》

- 吾王剑之所指，吾等心之所向。——《FATE/ZERO》

- 水晶帘动微风起，满架蔷薇一院香。——唐末将领高骈《山亭夏日》

- 落花人獨立，微雨燕雙飛。——晏幾道《临江仙·梦后楼台高锁》

- 醉後不知天在水，滿船清夢壓星河。——唐珙《题龙阳县青草湖》

- 林深时见鹿,海蓝时见鲸。
